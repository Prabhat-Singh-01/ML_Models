# 📊 Customer Churn Prediction using SVM

## 🚀 Project Overview

This project focuses on predicting customer churn using Machine Learning techniques. Customer churn refers to the situation where customers stop doing business with a company. Accurately predicting churn helps businesses take proactive steps to retain customers.

In this project, a **Support Vector Machine (SVM)** model is built and deployed to classify whether a customer is likely to churn or not.

---

## 🎯 Objectives

* Analyze customer data to identify churn patterns
* Build a predictive model using SVM
* Deploy the model using a simple web interface
* Help businesses improve customer retention strategies

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Platform:** Jupyter Notebook
* **Libraries Used:**

  * pandas (data manipulation)
  * matplotlib (data visualization)
  * scikit-learn (model building & evaluation)
  * joblib (model saving/loading)
  * streamlit (deployment)

## 🔍 Workflow

1. **Data Collection & Loading**

2. **Data Preprocessing**

   * Handling missing values
   * Encoding categorical variables
   * Feature scaling

3. **Exploratory Data Analysis (EDA)**

   * Visualizing churn distribution
   * Identifying key influencing factors

4. **Model Building**

   * Algorithm used: Support Vector Machine (SVM)
   * Data split into training & testing sets

5. **Model Evaluation**

   * Accuracy score
   * Confusion matrix
   * Classification report

6. **Model Saving**

   * Saved using `joblib`

7. **Deployment**

   * Built an interactive UI using Streamlit
